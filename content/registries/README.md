# /registries

Machine-readable protocol registries for the v0.1 draft:

- [`scopes.md`](scopes.md) — mandate scope registry v1 and matching rules
- [`error-codes.md`](error-codes.md) — `Ajar-Error-Code` registry and RFC 9457 profile
- [`settlement-adapters.md`](settlement-adapters.md) — registered settlement adapter identifiers

Registry changes follow the AEP process once v0.1 ships. Additions must include semantics, examples, and (for scopes) matching-rule notes.
